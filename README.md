## ü™ú **Steps Involved**

1. **Data Collection & Import**

   * Gather financial data from Excel sheets, databases, or CSV files.
   * Import data into Power BI Desktop.

2. **Data Cleaning & Transformation**

   * Use **Power Query Editor** to clean, merge, and structure raw data.
   * Handle missing values and ensure data consistency.

3. **Data Modeling**

   * Create relationships between tables (Fact & Dimension tables).
   * Build a star schema for optimized performance.

4. **DAX Calculations**

   * Develop **measures and calculated columns** for metrics like:

     * Total Revenue(Sales)
     * Gross Profit
     * Net Profit
     * Operating Profit
     * EBITDA
     * PBIT
     * Profit Margin

5. **Dashboard Design**

   * Build visuals such as bar charts, KPI cards, line charts, and bar charts.
   * Add slicers for period, region, or department filters.
   * Use bookmarks and tooltips for interactive navigation.

6. **Performance Analysis & Insights**

   * Compare **actual vs. target** financial results.
   * Identify Sales trends, revenue growth, and profitability ratios.

7. **Report Publishing & Sharing**

   * Publish dashboards to **Power BI Service**.
   * Enable **scheduled data refresh** and secure sharing.

## üìä **Result / Outcome**
